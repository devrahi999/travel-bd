1. Product Overview
Travel BD is Bangladesh's first fully integrated travel super-app. It connects travelers, business owners (hotels, restaurants, tour guides), and tour companies into one unified ecosystem.
1.1 Vision
"বাংলাদেশের প্রতিটি কোণ আবিষ্কার করুন — পরিকল্পনা থেকে booking পর্যন্ত সব এক জায়গায়।"
1.2 Core Value Proposition
Travelers: AI-powered personalized trip planning, all services bookable from one app
Business Owners: Digital presence, booking management, revenue analytics
Platform: Transaction commission, premium listings, data insights

2.1 Dual Account Support
A single user can be both a Traveler AND a Business Owner simultaneously. The app will have a "Switch Mode" button — tapping it switches the UI between Traveler Mode and Business Mode without requiring a separate login. Same account, same credentials, different interface.

15. Security & Data Privacy
Measure
Implementation
Authentication
JWT Token (short expiry) + Refresh Token rotation
API Security
Rate limiting, CORS whitelist, HTTPS only
Data Encryption
Sensitive data encrypted at rest (Supabase)
Document Storage
Business docs in private Cloudflare R2 bucket (not public)
Payment Security
PCI-compliant gateway, zero card data stored on our servers
Input Validation
Server-side validation on all inputs, parameterized queries (SQL injection prevention)
User Data Rights
Data export + account deletion option available in Settings
